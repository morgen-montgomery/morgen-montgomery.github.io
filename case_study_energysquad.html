Description

As a PM for a SaaS Company in the Green Energy space, I spent a fair amount of time doing reasearch into current processes. I would create User Stories and identify deliverables for different in-house developed applications. One in particular was a Photo Application, where a verbose amount of photo requirements needed to be defined in an orderly fashion.

Technologies

The end product was developed using Angular and Amazon's AWS platform. The requirements shown below were mapped out using Mindmeister- a lightweight mind-mapping software.
