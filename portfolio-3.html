CONFIDENT BAGEL

Description

Confident Bagel is a web application that runs a Sentiment Analysis on any given Twitter hashtag and displays visually appealing results.

Technologies

Confident Bagel was developed using Python, JavaScript, Node, Express, D3.js, PostgreSQL, Knex, Twitter's text mining API and Jade/Pug.

Live at: https://shrouded-refuge-87693.herokuapp.com/

GitHub: https://github.com/morgen-montgomery/Twitter_Sentiment_Analysis.git
